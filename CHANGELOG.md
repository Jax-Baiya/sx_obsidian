# Changelog

All notable changes to this project are documented in this file.

## [1.0.0] - 2026-02-28

### Added
- Source-scoped backend, API, and operational control-plane integration.
- Modularized Obsidian plugin library interactions and metadata workflows.
- TUI workflows for diagnostics, import, and plugin deployment.

### Improved
- CI reliability with shell lint/format gates, shell tests, and expanded Python regression coverage.
- Release and runbook documentation for setup, migration, and operational recovery.

### Notes
- This is the first professional stable release of the SX Obsidian plugin stack (`sx-obsidian-db`).
